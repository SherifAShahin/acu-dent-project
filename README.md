# ACU Dental AI - نظام فحص الأسنان الذكي

هذا المشروع هو مشروع تخرج لكلية الهندسة بجامعة الأهرام الكندية (ACU)، وهو نظام هجين متطور لفحص أمراض الأسنان باستخدام الذكاء الاصطناعي.

## التشغيل على VS Code (Local Setup)

اتبع الخطوات التالية لتشغيل المشروع على جهازك الشخصي:

### 1. المتطلبات (Prerequisites)
*   تثبيت **Node.js** (إصدار 18 أو أحدث).
*   تثبيت **VS Code**.

### 2. التثبيت (Installation)
1.  قم بتحميل ملفات المشروع وفتح المجلد في VS Code.
2.  افتح Terminal جديد في VS Code وقم بتثبيت المكتبات المطلوبة:
    ```bash
    npm install
    ```

### 3. إعداد مفتاح API (Environment Variables)
1.  قم بإنشاء ملف جديد باسم `.env` في المجلد الرئيسي للمشروع.
2.  انسخ المحتوى من `.env.example` وضعه في ملف `.env`.
3.  أضف مفتاح Gemini API الخاص بك:
    ```env
    GEMINI_API_KEY="ضع_مفتاحك_هنا"
    ```
    *يمكنك الحصول على المفتاح مجاناً من [Google AI Studio](https://aistudio.google.com/app/apikey).*

### 4. تشغيل المشروع (Running the App)
قم بتشغيل خادم التطوير:
```bash
npm run dev
```
بعد التشغيل، سيظهر لك رابط (غالباً `http://localhost:3000`)، قم بفتحه في المتصفح.

## مميزات المشروع (Project Features)
*   **نظام هجين (Hybrid AI)**: يجمع بين قوة Gemini API ونماذج مدربة خصيصاً على Google Colab.
*   **تحليل فوري**: اكتشاف التسوس، التهابات اللثة، وتصبغات الأسنان في ثوانٍ.
*   **واجهة احترافية**: مصممة خصيصاً لجامعة الأهرام الكندية تحت إشراف د. أمين السعيد.

## هيكل المشروع (Project Structure)
*   `server.ts`: خادم Express (Backend).
*   `src/App.tsx`: واجهة المستخدم (Frontend).
*   `vite.config.ts`: إعدادات بناء المشروع.
